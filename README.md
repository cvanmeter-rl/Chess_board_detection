# Automated Chess Move Recommendation via Deep Learning-Based Board and Piece Recognition
## Steps To Run Program
1. Clone Repository
2. Download The Following Required Libraries/Packages:
   - PyQt5
   - Pillow
   - numpy
   - torchvision
   - torch
   - ultralytics
   - stockfish
   - skimage
   - cv2
   - pyautogui
3. Run NewUI.py and you are all set
